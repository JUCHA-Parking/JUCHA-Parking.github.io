데이터 삭제 요청

InnoByte Note 계정과 관련 데이터 삭제를 요청하실 수 있는 두 가지 방법을 안내합니다. 두 방법 모두 동일한 결과를 가져오며, 어느 쪽을 사용하셔도 됩니다.

Method 01 · Recommended

앱 내에서 직접 탈퇴 (권장)

가장 빠르고 즉시 처리되는 방법입니다. 앱이 설치되어 있고 로그인된 상태라면 다음 절차를 따라주세요.

  1. InnoByte Note 앱을 실행하고 로그인합니다.
  2. 좌측 사이드바 하단의 설정 버튼을 누릅니다.
  3. 회원 탈퇴 메뉴를 선택합니다.
  4. 안내 메시지를 읽고 본인 이메일을 입력하여 확인합니다.
  5. 탈퇴 완료 — 모든 데이터가 즉시 삭제됩니다.

01삭제되는 데이터

탈퇴 처리 시 다음 데이터가 영구적으로 삭제되며, 한 번 삭제된 데이터는 복구할 수 없습니다.

항목 저장 위치 삭제 여부
계정 정보 (이메일, 이름, Google ID) 자체 서버 + Firebase 즉시 삭제
인증 토큰 (access·refresh) 자체 서버 즉시 무효화
노트, 손글씨, 첨부 PDF 메타데이터 자체 서버 즉시 삭제
PDF 검색용 벡터 임베딩 Pinecone 즉시 삭제
퀴즈 세트 및 학습 기록 자체 서버 즉시 삭제
STT 음성 스트림 저장하지 않음 애초에 미저장
접속 로그 자체 서버 최대 3개월 후 자동 삭제
이용자 본인의 Google Drive에 저장된 PDF 파일 이용자 Google Drive 삭제하지 않음

Google Drive 파일에 대한 안내

본 서비스는 이용자의 Google Drive에 PDF를 보관하지만, 해당 파일은 이용자 본인의 자산입니다. 따라서 회원 탈퇴 시에도 Drive에서 자동으로 삭제하지 않습니다. Drive 파일까지 삭제하길 원하시면 drive.google.com에서 NoteAI 폴더를 직접 삭제해 주세요.

02처리 기간

  • 앱 내 탈퇴 — 즉시 처리, 평균 3초 이내
  • 이메일 요청 — 영업일 기준 7일 이내 처리, 처리 완료 시 회신

관련 법령에 따라 보존이 의무화된 정보(통신비밀보호법상 로그 등)는 법정 기간 동안 분리 보관 후 폐기됩니다.

03주의 사항

  • 탈퇴 후 동일한 Google 계정으로 재가입은 가능하지만, 이전 데이터는 복구되지 않습니다.
  • 중요한 노트는 탈퇴 전에 설정 → PDF 추출을 통해 백업해 두시길 권장합니다.
